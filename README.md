**哈希分析大师 5.0** 是一款专为区块链数据分析师量身打造的 **波场 (TRON) 哈希实时可视化与趋势预测系统**。

本项目基于 **React 19** + **TypeScript** + **Tailwind CSS** 构建，采用无后端（Serverless）架构，直连 TronGrid 主网节点。系统集成了 **多维走势图表**、**全域长龙雷达**、**v4.0 稳定版 AI 演算引擎** 以及 **专业级沙盘模拟系统**，旨在提供毫秒级的数据响应与高置信度的决策支持。

## 🌟 核心功能详解 (Core Features)

### 1. 🎮 专业沙盘模拟系统 (Simulation Sandbox)
**不仅仅是记录，更是策略验证的战场。**
系统内置了一个功能完备的资金管理演练场，支持 **全自动托管** 与 **手动干预**，所有数据本地持久化存储。

*   **📊 动态资金管理 (Dynamic Capital)**：
    *   **实时重置机制**：在设置中修改“初始本金”时，系统会立即重置当前账户余额与最大回撤记录，方便用户随时以新的基准进行测试（例如：将本金从 1w 改为 2w，收益率即刻归零重算）。
    *   **盈亏曲线全景图**：支持 **拖拽与缩放** 的交互式资金曲线图，毫秒级记录每一笔下注后的资金变动，精准捕捉最大回撤点（Max Drawdown）。
    *   **多维统计**：实时计算胜率、盈亏比、最大连红/连黑、最高收益（Max Profit）等关键指标。

*   **🤖 AI 智能托管 (AI Auto-Hosting)**：
    *   **AI 动态凯利 (AI Kelly)**：独家创新策略。结合 AI 实时输出的置信度 ($p$) 与赔率 ($b$)，利用凯利公式 $f^* = \frac{bp - q}{b}$ 动态计算最优下注比例。支持调节“风险系数”（如 20% Kelly），在攻击与防守间取得平衡。
    *   **全域全规则扫描 (Global Full Scan)**：系统自动在后台轮询所有已配置的采样规则（如 30秒、1分、5分等），寻找置信度 **>94%** 的最佳入场点。即“狙击手模式”，不开枪则已，一开枪必中。

*   **📈 经典倍投策略库**：
    *   内置 **马丁格尔 (Martingale)**、**达朗贝尔 (D'Alembert)**、**斐波那契数列**、**1-3-2-6**、**Paroli** 等经典模型，并支持自定义倍数序列。

### 2. 🧠 AI 稳定演算矩阵 v4.0 (AI Prediction Matrix)
**告别模棱两可，追求极致的确定性。**
内置四大核心统计算法，并引入 **强制互斥逻辑 (Mutual Exclusion)**，在同一区块高度下，单双与大小之间自动择优，仅输出置信度最高的一个结果。

*   **四大核心模型**：
    1.  **马尔可夫状态迁移 (Markov Chain)**：捕捉 1-2 跳等震荡规律，计算状态转移概率矩阵。
    2.  **贝叶斯后验推理 (Bayesian Inference)**：基于大数定律，评估当前分布相对于理论期望的 3σ 极值偏差。
    3.  **频谱周期律检测 (Spectral Analysis)**：利用离散傅里叶变换 (DFT)，破译伪随机生成器中的固定步长循环（如每 7 期反转）。
    4.  **密集簇群共振 (Density Clustering)**：识别微观窗口内的能量爆发点与聚类特征。

*   **熵值风控 (Entropy Score)**：
    *   系统实时计算当前哈希流的信息熵。当熵值过高（>40）时，代表市场处于极度混乱状态，AI 将自动锁定，拒绝给出预测建议，从而避免无效损耗。

### 3. 🐉 全域长龙雷达 (Dragon Radar)
**全网扫描，不错过任何一条长龙。**

*   **多维监控**：
    *   **走势长龙**：监测传统大路图的纵向连出（如：单单单单）。
    *   **珠盘行龙**：监测珠盘路特定 **物理行**（如第3行）的横向连出。严格遵循 `(高度 - 偏移) / 步长 % 行数` 的物理对齐逻辑。
*   **智能交互**：
    *   **我的关注**：点击“爱心”即可收藏特定规则的特定趋势，置顶显示。
    *   **一键跳转**：点击长龙卡片，系统自动切换图表视图至该规则，实现从发现到分析的无缝衔接。

### 4. 📊 极速图表引擎 (Visualization Engine)
*   **大路走势图 (Big Road)**：经典红蓝走势复刻，智能折行，支持超长历史数据回溯。
*   **珠盘路 (Bead Road)**：精准还原澳门标准珠盘布局，支持 6行/10行 自定义布局。
*   **多维采样**：支持自定义 **区块步长** (Step) 与 **起始偏移** (Offset)，可同时运行数百条规则。

## 🛠️ 技术架构与性能 (Tech Stack)

*   **前端框架**：React 19 (Strict Mode) - 利用最新的并发渲染特性，确保在大数据量下的界面流畅度。
*   **语言**：TypeScript - 强类型约束，确保金融级计算逻辑的严谨性，杜绝类型错误。
*   **样式引擎**：Tailwind CSS - 原子化 CSS，实现极致的响应式布局与美学设计。
*   **图标库**：Lucide React - 轻量级、风格统一的 SVG 图标集。
*   **数据源**：TronGrid API - 直连波场主网节点，支持 WebSocket 级的高频轮询。
*   **持久化**：LocalStorage - **零后端设计**，所有配置、API Key、模拟记录均存储在用户本地浏览器中，最大程度保护用户隐私与资金安全。

## 📖 快速上手指南 (Quick Start)

### 1. 环境配置
初次进入系统，请点击右上角 **设置 (Settings)** 图标：
- 输入您的 **TronGrid API Key** (推荐使用付费 Key 以获得更高频的同步速率，免费 Key 可能会有 1-2 秒延迟)。
- 可自定义 **主题配色** (单/双/大/小 的显示颜色)。

### 2. 规则管理
- 使用 **批量编辑** 功能，通过 CSV 格式快速导入几十条采样规则。
- 格式示例：`规则名,步长,偏移,走势行数,珠盘行数,龙提醒阈值`
- 例如：`30秒极速,10,0,6,6,3`

### 3. 实战演练
- **观察**：进入 **AI 数据预测** 页面，观察“全域实时流水”，寻找熵值低、模型共振的信号。
- **验证**：进入 **模拟下注** 页面，创建“AI 全域全规则”托管任务，设定好止损线 (Stop Loss)，体验自动化策略的运行效果。
- **复盘**：点击模拟页面的“总盈亏曲线”，拖拽查看资金回撤详情，优化您的下注策略。
