# 波场 (TRON) 哈希实时分析大盘 5.0 稳定版

本平台是一款专为波场区块链（TRON Network）深度分析师打造的实时数据可视化与趋势预测大盘。基于 React 19 + TypeScript + Tailwind CSS 构建，旨在提供军工级的数据稳定性与毫秒级的趋势响应。

---

## 💎 5.0 版本核心特性

### 1. 🧬 多步长对比共振模式 (Multi-Rule Resonance)
- **勾选对比**：支持同时勾选多个步长规则（如“单区块”与“20区块”），图表自动并排展示，方便寻找不同步长下的走势共振机会。
- **自适应布局**：在“单双/大小走势”及“珠盘路”页面，多选规则时系统自动切换为双列响应式排版 (lg:grid-cols-2)，极大提升对比分析效率。

### 2. 🧠 全域 AI 稳定演算引擎 (Global Background AI)
- **永不停歇的预测**：核心 AI 引擎已升级为全局后台运行模式。无论您停留在哪一个标签页（实时大盘、长龙榜、甚至在配置页面），AI 引擎都会在后台持续捕获所有采样规则的区块对齐，并进行深度概率演算。
- **四维模型共振**：集成马尔可夫链、贝叶斯后验、频谱周期律及密集簇群共振模型，置信度 >92% 时自动输出确定性信号。

### 3. 🐉 矩阵级长龙雷达与批量配置
- **全采样扫描**：针对所有定义的采样规则进行实时毫秒级扫描，捕捉走势与珠盘行的极致长龙。
- **一键阈值同步**：新增“全规则长龙阈值批量设置”，可在设置面板一键同步所有规则的预警连出数。
- **智能跳转**：点击龙榜卡片，系统会瞬间切换至对应规则的分析面板。

## 🛠️ 技术规格与架构

- **前端框架**：React 19 (Strict Mode)
- **样式引擎**：Tailwind CSS (响应式布局，完美适配移动端与超宽屏)
- **核心逻辑**：
  - **全局后台分析器**：利用 React Effect 与 Ref 维持不中断的 AI 计算流水线。
  - **物理对齐算法**：确保不同标签页之间的数据对齐 100% 一致，解决计算抖动。
- **数据源**：TronGrid API (支持 TRON-PRO-API-KEY 鉴权)

## 📖 实战操作指南

### 第一步：配置核心引擎
在“设置”面板中输入您的 **TronGrid API Key**。
> *建议：使用具有付费配额的 Key 以支持每秒一次的高频轮询。*

### 第二步：开启对比模式
在页面顶部的“对比观察勾选框”中选中您想要关注的步长规则匹配。系统会自动加载历史数据并为您并排展示走势。

### 第三步：AI 信号跟随
切换至“AI 数据预测”页面查看全域历史。由于 5.0 采用后台运行，您切回该页面时将看到最新的、已对齐的预测结果，无需重新等待。
